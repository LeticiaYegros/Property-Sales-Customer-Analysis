# Property Sales & Customer Analysis

## Project Overview

This project analyses property sales data using Microsoft Excel, with a focus on customer activity, sales performance and revenue distribution across property categories.

The workbook combines pivot tables, slicers and reporting techniques to explore sales trends, compare product categories and evaluate representative performance over multiple years.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Lookup Functions

## Data Preparation

The dataset was reviewed and structured before analysis by:

- Organising transactional records
- Verifying consistency across related tables
- Linking datasets using unique identifiers
- Creating supporting calculations
- Preparing data for reporting and comparison

## Analysis Included

- Product sales trends
- Representative performance analysis
- Customer activity review
- Revenue distribution by property category
- Regional sales comparisons
- Year-over-year sales tracking

## Dashboard Features

- Interactive filters and slicers
- Multi-year sales comparison
- Product category breakdowns
- Representative ranking analysis
- Customer-focused reporting views

## Files Included

- Excel workbook
- Dashboard screenshots
- Key insights worksheet
- Project documentation

## Dataset

The dataset used in this project contains fictional property sales records and was used for analysis, reporting and dashboard development practice.
